# 常用插件

# clean-webpack-plugin

清理项目打包目录

# HtmlWebpackPlugin

自动生成一个HTML文件，并将打包后的bundle自动插入HTML中。支持读取模板。

# mini-css-extract-plugin

style-loader处理过后的css代码是直接以style标签的形式插入head内，这时候如果需要将css代码分离到一个单独css文件中，就需要此插件。

此插件暂不支持HMR，所以应该在生产环境中使用。
