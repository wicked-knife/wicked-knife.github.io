# 搭建开发环境

进行项目开发时，如果每次改变并保存代码浏览器能够自动刷新的话，这样可以极大地提高开发效率与体验。

安装webpack-dev-server

npm i -D webpack-dev-server

在webpack.common.js中作如下配置:

module.exports = {
    ...
    devServer: {
      contentBase: 'dist',
      open: true,
      hot: true,
      overlay: true,
      proxy: {
        ...
      }
    }
}

contentBase: 指webpack-dev-server以哪个目录启动服务。

open: 自动打开浏览器。

hot: 模块热更新。(后续会说明)

overlay: 打包报错时会在浏览器出现弹窗，并打印错误警告。

proxy：主要用来解决前后端分离时的跨域问题，用作代理。具体可参考官方文档。

在package.json中配置:

{
  ...
  "scripts": {
    ...
    "dev": "webpack-dev-server --config webpack.config.js"
    ...
  }
  ...
}

npm run dev即可打开本地服务进行项目开发。
